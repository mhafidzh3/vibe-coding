# Vibe Coding Project

A backend project using Bun, ElysiaJS, Drizzle ORM, and PostgreSQL.

## Features
- **Bun**: Fast JavaScript runtime.
- **ElysiaJS**: High-performance web framework.
- **Drizzle ORM**: Type-safe SQL ORM.
- **PostgreSQL**: Robust relational database.

## Getting Started
1. Install dependencies: `bun install`
2. Set up `.env`: Copy `.env.example` and fill in `DATABASE_URL`.
3. Push schema: `bun run db:push`
4. Run dev: `bun run dev`
